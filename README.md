# osu! Discord Statistics Bot

A modular, event-driven Discord bot built in Java designed to fetch and display real-time player statistics from the official osu! API. 

This project serves as a practical portfolio piece to demonstrate clean code principles, API integration, and asynchronous programming in Java.

## 🎯 Project Goals & Motivation
As an active osu! player and a student currently expanding my programming knowledge, I created this bot to combine my passion for the game with my desire to level up my Java skills. 

While the bot is a fun utility for the community, the main technical objective of this project is to bridge the gap between theoretical computer science concepts and practical application. My focus areas include:
* **API Integration:** Handling secure OAuth2 authentication and parsing complex, nested JSON responses.
* **Asynchronous Execution:** Managing non-blocking I/O requests within the Discord gateway.
* **Clean Architecture:** Designing maintainable, decoupled components (separating the API service logic from the Discord listener).

## 🛠️ Tech Stack
* **Language:** Java 
* **Framework:** JDA (Java Discord API)
* **HTTP Client:** OkHttp3
* **JSON Parser:** Google Gson
